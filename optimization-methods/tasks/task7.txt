Задание:
Написать функцию nsearch(f, df, x0, tol) реализующую поиск методом Ньютона. 

Входные переменные
• f  — целевая функция,
• df —  производная целевой функции,
• x0 — стартовая точка,
• tol — заданная точность.
Две целевые функции и их градиенты заданы аналитически, это функции Химмельблау и Розенброка.

Выходная переменная
• answer_ = [xmin, fmin, neval, coords]

где:
• xmin —  точка минимума, 
• fmin —  значение целевой функции в точке минимума,
• neval —  число вычислений производной целевой функции,
• coords —  массив координат x во время оптимизации.

Пояснение
В качестве критерия продолжения поиска используйте
    (norm(deltaX) >= tol) and (k < kmax),
где kmax = 1000 — максимальное число итераций, deltaX —  разница между двумя точками, найденными на соседних итерациях.

Для формирования массива coords воспользуйтесь конструкцией
    coords.append(x)

Для обращения матрицы -H0 и умножения ее на градиент g0 воспользуйтесь библиотечной функцией
    dx = np.linalg.lstsq(-H0, g0)[0],
где
• dx - разница между новым и текущим значением x,
• H0 - значение матрицы Гессе в текущей точке,
• g0 - значение градиента в текущей точке.

Для поиска матрицы Гессе напишите функцию 
    H(x0, tol, df),
которая будет возвращать матрицу Гессе, принимая на вход:
• x0 — точку, в которой ищется матрица Гессе.
• tol — заданная точность, из которой находится дельта для производной deltaX = 0.1 * tol.
• df — градиент (первая производная).

Используйте формулу центральных разностей.
